REST API TASKS -- JSON -- Good Practices

GET /users --> HTML Document Render (done)
GET api/users --> List all users as JSON (Done)

GET /user/1 --> Get the user with ID 1 (DONE)
GET /user/2 --> Get the user with ID 2 (DONE)

Dynamic Path Parameters
GET /api/users/:id --> Variable | Dynamic

POST /users --> Create a new user

PATCH /user/1 --> Edit the user with ID 1

DELETE /user/1 --> Delete the user with ID 1
